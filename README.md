# Painting CRM

CRM system for Black Line Painting LLC.

## Purpose
Track leads, estimates, jobs, and pipeline from initial contact to 5-year warranty follow-up. Built to support scaling from $200k to $400k/year.

## Core Entities
- **Leads**: Name, phone, email, address, source, city (Bremerton, Poulsbo, Silverdale, Port Orchard, Bainbridge Island, Gig Harbor, Seabeck), status
- **Estimates**: Linked to lead, line items, materials, labor, margin %, status (draft/sent/approved/declined)
- **Jobs**: Scheduled dates, crew assignment, warranty start date
- **Customers**: Post-job contact info, warranty tracking
- **Pipeline**: Stages - New Lead → Contacted → Estimate Sent → Approved → Scheduled → In Progress → Completed → Warranty

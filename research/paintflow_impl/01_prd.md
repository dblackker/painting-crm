# PaintFlow PRD
**Version:** 0.2  
**Date:** 2026-05-17

## 1. Vision
One app for solopreneur painters to run leads → estimates → jobs → invoices → profit tracking.

## 2. ICP
1-3 person crews, $200-600k revenue, phone-first, currently using Jobber + spreadsheets.

## 3. Problems
- 30% leads lost to no follow-up
- Don't know job profitability until tax time
- Too many subscriptions
- Estimating takes too long

## 4. MVP User Stories

### Onboarding
- Sign up with magic link
- Create org, invite 1 crew
- Import Jobber CSV or skip
- See sample lead + estimate
- Time to first estimate <10 min

### Leads
- Add lead manually or embed form
- Pipeline board: New, Contacted, Estimate Sent, Won, Lost
- Tap to call/text

### Estimating
- Good/Better/Best packages
- Production rates: sq ft, linear ft
- Add photos
- Send via SMS/email link
- Client accepts with e-sign

### Follow-up Drips
- Auto enroll on estimate sent
- Day 1 SMS, Day 3 Email, Day 7 SMS
- Stop on accept/decline
- 2-way SMS inbox

### Scheduling
- Week calendar view
- Drag job to day
- Google Calendar sync

### Invoicing
- Create from estimate
- Stripe Checkout, 50% deposit
- Mark paid manually

### Job Costing
- Budget auto from estimate
- Log time + expenses with receipt photo
- Real-time margin %

### Customer Portal
- Public link to view/accept/pay estimate

## 5. Success Metrics
- 10 paying orgs in 30 days
- 70% estimate close rate
- 80% jobs have costing data
- <10 min time-to-first-estimate

## 6. Out of Scope MVP
- QuickBooks 2-way sync
- Review requests
- Offline PWA sync
- Crew mobile app

## 7. Pricing
Solo $49/mo, 1 user + 1 crew, unlimited leads. Launch $29/mo for first 20.
